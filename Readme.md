# Ozone Fitness - Gym Management System

A full-stack web application designed to manage gym memberships, facilitate class bookings, and collect user feedback.

## Key Features
* **User Registration:** Seamless onboarding for new gym members.
* **Virtual Training:** Dedicated section for guided workout routines (Biceps, Triceps, Cardio).
* **Payment Integration:** Simulated payment gateway and transaction ID tracking.
* **Feedback Loop:** PHP-based feedback system storing user reviews in a MySQL database.
* **Responsive Design:** Utilizes Tailwind CSS and custom styling for a modern look.

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP (Procedural)
* **Database:** MySQL
* **Server:** Apache (XAMPP/WAMP)

## How to Run Locally
1. Install **XAMPP** or **WAMP**.
2. Clone this repository into your `htdocs` folder.
3. Open **phpMyAdmin** and create a database named `gym`.
4. Import the file located in `database/gym_backup.sql`.
5. Open your browser and navigate to `localhost/Ozone-Fitness-Management`.
